# 🚀 Quick Deploy Guide

## 1️⃣ Docker (Eng Oson)

```bash
docker-compose up -d
```

## 2️⃣ Heroku

```bash
heroku create navohub-bot
heroku config:set BOT_TOKEN="your_token"
git push heroku main
```

## 3️⃣ Railway

```bash
railway init
railway up
```

## 4️⃣ VPS

```bash
python3 bot.py
```
